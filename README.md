<!-- GETTING STARTED -->
## Getting Started
### Prerequisites

_First of all, you need to register on the Kwork._

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Druidae/SKwork.git
   ```
2. Install pip packages
   ```sh
   pip install -r 'requirements.txt'
   ```
3. Enter your **Login**, **Password** in `.env`
